# Quadrinho sobre a conscientização do meio ambiente

## 📒 Descrição
Projeto idealizado por duas IA's (Inteligências Artificais Generativas) na qual a ferramente Bing CHAT escreveu um roteiro sobre um quadrinho que relata a história de um menino
que acabou se dando mal após jogar lixo na rua

## 🤖 Tecnologias Utilizadas
IA Generativa BING CHAT para a realização do roteiro
IA Generativa Leonardo AI para a criação das imagens dos quadrinhos
Photoshop para a edição dos balões

## 🧐 Processo de Criação
Utilizei o "ChatGPT" para revisar minha pesquisa sobre IAs Generativas, estruturar o conteúdo e refinar minhas ideias. Por outro lado, o "Leonardo AI" foi usado para gerar imagens exclusivas para o meu quadrinho
